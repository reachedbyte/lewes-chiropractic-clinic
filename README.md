# Lewes Chiropractic Clinic

Premium, mobile-first website for Lewes Chiropractic Clinic — established 1990, central Lewes, East Sussex.

## Stack
- Static HTML / CSS / vanilla JS — no build step
- Cormorant Garamond + Inter typography
- Editorial luxury design system

## Run locally
Open `index.html` in a browser, or serve the directory with any static server:
```
npx serve .
```

## Deploy
Deployed via GitHub Pages from the `main` branch.
